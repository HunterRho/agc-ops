Setup Guide & Architecture
Recommended Stack — Production Setup

This dashboard is a fully functional front-end prototype. To make it production-ready with cloud storage, invoice upload, and multi-device access, deploy it with the following stack:

DATABASE — Google Sheets (Free)

Create a Google Sheet with these tabs: Transactions, Inventory, Adjustments, SpotPrices, Contacts. Use Google Apps Script to expose a Web App endpoint that this dashboard POSTs data to. Data persists forever, accessible from any device via Google account. Tim can view/edit directly in Sheets.

INVOICE STORAGE — Google Drive

Each transaction row contains a Google Drive file link. The Apps Script upload handler accepts a base64-encoded file and stores it in a "Invoices/YYYY-MM/" folder structure. Invoices are accessible by link without sharing publicly.

SPOT PRICES — Metals API

Use metals.dev or goldapi.io (free tiers available) to auto-populate spot prices at transaction entry time. The script pulls current AU/AG/PT/PD and saves it with each transaction row.

HOSTING — GitHub Pages or Google Sites

Host this HTML file on GitHub Pages (free, private repo possible) or serve it from a Google Site. The dashboard calls the Apps Script Web App URL for all data operations. Zero server costs, zero maintenance.

Compliance & CTR Tracking

Each transaction includes a CTR_Required flag. The system auto-flags any cash transaction or same-day combination of transactions over $10,000 with a single party. These entries are tagged and filterable for Tim's end-of-month compliance export. The Contacts sheet tracks lifetime transaction totals per customer for BSA reporting awareness. This does NOT replace your legal compliance program — it supplements it.


Spot % Calculation: (Price Paid ÷ (Troy Oz × Spot Price)) × 100. For BUY transactions, lower % = better margin. For SELL transactions, higher % = better premium captured.


Gross P/L on Buys: Not calculated at entry — recorded as COGS. P/L is realized when the item sells. For bullion with known cost basis, P/L = (Sell Price − Buy Price) × Troy Oz.
